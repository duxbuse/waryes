**Deck Building Mechanics:**
- **50 Activation Points** maximum per deck
- **Progressive Slot Costs**: Each category has escalating costs per slot
  - Example (Infantry Division): INF slots cost [1,1,1,1,2,2,3,3], TNK slots cost [2,3,4,5]
- **Unit Availability**: Cards give X units (e.g., "4x Guardsmen" or "2x Leman Russ")
- **Veterancy Trade-off**: Higher vet = fewer units per card
- **Commander Limit**: Max 1-2 commander cards per deck
- **Transport Selection**: Popup for infantry to choose transport type

- **Slot Types**:

- **Minimum**: At least 1 player per team required

- Drag on minimap to pan camera

- Unit icons match their category (same as tactical view icons)
- Selected units highlighted with white outline
- Capture zone colors update as ownership changes

**Kill Tracking Requirements:**
- Track every kill with: killer unit ID, victim unit ID, victim cost, timestamp
- For infantry squads, track partial losses (models killed vs full squad wipes)
- "Kill Value" = sum of all victim unit costs killed by this unit
- Partial squad kills count proportionally (4/10 killed = 40% of unit cost)

**Hero of the Match:**
- Unit with highest total "Kill Value" (sum of enemy costs destroyed)
- Displayed prominently with kill count and total value

**Most Cost Effective:**
- Unit with highest ratio: (Kill Value) / (Unit's own cost)
- Efficiency shown as multiplier (e.g., "8.5x" means killed 8.5x their own cost)
- Minimum 1 kill required to qualify

**Unit Loss Tracking:**
- Record what destroyed each unit (enemy unit name or "Artillery", "Routing", etc.)
- For partial infantry losses, show (X/Y) format
- Timestamp when the loss occurred
- "Routing - fled map" for units that routed off the battlefield

**Status Column Values:**
- ALIVE - Full health/squad
- ALIVE (X/Y) - Partial squad remaining
- DESTROYED - Fully destroyed
- ROUTING - Currently routing (fled or about to flee)
